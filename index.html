Utilização de layout liquido: adaptando várias resoluções.
Criando div id container.
Quando estiver na pasta CSS e for inserir imagem da pasta IMG, deve utilizar ../ para subir o nível, ou seja, ir para a pasta principal para depois entrar na pasta imagem.
Utilizando o * no CSS, você pode configurar toda a página, limpando.
No CSS, quando utiliza a tag body, pode definir a fonte, a cor de fundo, inserir imagem de fundo.
No CSS, quando utilizar tag não precisa de nenhum outro simbolo antes, já quando for se ferir a um id do html, deve colocar # antes e no caso de class do html, deve inserir o . (ponto)

CSS: configuranção do topo:
- Cor de fundo;
- Altura: utilizando a unidade 'em', pois caso aumento o tamanho do texto, essa altura será proporcional ao tamanho do texto;
- min-height define o tamaho mínimo do topo.
